What does CSS stand for?

Cascading Style Sheets.

What is the purpose of the "box-sizing" property in CSS?

It defines how the total width and height of an element is calculated.

What is the difference between "margin" and "padding" in CSS?

"Margin" is the space outside an element, while "padding" is the space inside an element.

What does the term "pseudo-class" refer to in CSS?

It is used to define a special state of an element, such as when it is hovered over or selected.

What is the CSS property used to control the transparency of an element?

Opacity

What is the CSS unit that represents a percentage of the viewport width?

"vw" (viewport width).

What does the "float" property do in CSS?

It specifies how an element should float in relation to its surrounding elements.

What is the CSS selector that selects all direct child elements of a parent element?

"> (greater-than sign)".

What is the purpose of the "z-index" property in CSS?

It controls the stacking order of positioned elements.

What is the difference between "display: block;" and "display: inline;" in CSS?

"display: block;"

What is the CSS property used to add rounded corners to an element?

"border-radius"

What is the CSS property used to specify the font of an element?

"font-family"
